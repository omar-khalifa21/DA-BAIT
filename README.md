# Arabic AI Debate Platform

An anonymous, real-time debate platform inspired by Omegle — but with a twist.

Two users are randomly matched and given a topic to debate.  
An AI system listens to the conversation and evaluates both participants based on:

- Factual accuracy  
- Logical consistency  
- Argument structure  
- Language clarity (Arabic-focused)

At the end of the session, the AI declares a winner and provides feedback.

---

## Vision

To create a competitive, educational, and engaging space for Arabic speakers to:

- Practice critical thinking  
- Improve argumentation skills  
- Learn through debate  
- Compete in a fair, AI-judged environment  

---

## Features

### Real-time Matching
- Anonymous 1v1 pairing (Omegle-style)
- Topic-based or random debates

### AI Judge System
- Evaluates both sides of the debate
- Scores based on:
  - factual correctness
  - coherence
  - relevance
- Generates explanation for the verdict

### Arabic-First NLP
- Supports Arabic dialects and MSA
- Designed for Arabic LLMs or fine-tuned models

### Live Communication
- WebSocket-based real-time chat or voice
- Optional speech-to-text pipeline

### Scoring System
- Per-round scoring
- Player ranking system (future)
